# Tags Input

## Prompt (en-US)

Create a tags input for managing collections of text tags or labels. Feature tag creation via Enter or comma, tag removal with X button, duplicate prevention, and customizable tag display with colors and icons.

---

## 提示詞 (zh-CN)

### 組件描述
標籤輸入用於管理多個標籤或標記，適用於文章標籤、技能選擇、分類管理等場景。

### 視覺要點
- 標籤以藥丸形狀顯示
- 每個標籤帶刪除按鈕（X）
- 輸入框在標籤之後
- 可選：標籤顏色區分
- 標籤數量限制提示

### 互動設計
- Enter 或逗號創建標籤
- 點擊 X 刪除標籤
- Backspace 刪除最後一個
- 拖拽重新排序（可選）

### 變體風格
- **默認**：直觀創建、清晰移除、平滑互動
