# Range Slider

## Prompt (en-US)

Create a flexible range slider for selecting single or multiple values within defined ranges. Feature dual handles, custom step sizes, value labels, and smooth animations. Support vertical/horizontal orientations with keyboard accessibility.

---

## 提示詞 (zh-CN)

### 組件描述
範圍滑塊支持單值或區間選擇，適用於價格篩選、音量控制、亮度調節等場景。

### 視覺要點
- 軌道（track）和滑塊（thumb）
- 已選區間高亮填充
- 當前值標籤顯示
- 最小/最大值標記
- 可選：刻度線

### 互動設計
- 拖拽滑塊調整值
- 點擊軌道跳轉
- 鍵盤方向鍵微調
- 雙手柄區間選擇

### 變體風格
- **極簡垂直**：垂直佈局、清晰指示
- **玻璃態雙手柄**：毛玻璃背景、發光效果
- **Bootstrap價格**：價格篩選專用、藍色主題
- **Material亮度**：Material 風格、圖標裝飾
- **新擬物音量**：凹凸效果、柔和陰影
