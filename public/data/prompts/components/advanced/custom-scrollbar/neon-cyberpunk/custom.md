# Neon Cyberpunk Custom Scrollbar

## 中文版本 (zh-CN)

請創建一個 Neon Cyberpunk 風格的自定義滾動條組件。

### 設計理念

Neon Cyberpunk 滾動條靈感源自賽博朋克美學和霓虹燈文化，融合了未來主義、都市夜景和數字反烏托邦的視覺元素。設計哲學強調高對比度、發光效果和電子脈衝感，創造出充滿科技感和叛逆精神的視覺體驗。滾動條不僅是導航工具，更是界面中的能量指示器，象徵著數據流動和電子脈動。

### 視覺特徵

- **霓虹發光**: 滑塊使用鮮豔的霓虹色彩 (#00ffff 青色、#ff00ff 洋紅、#ffff00 黃色)，配合 box-shadow 實現多層次發光效果
- **電路板軌道**: 軌道採用深色背景 (#0a0a0a) 搭配電路紋理或掃描線效果，模擬電子電路板
- **多重光暈**: 滑塊使用 3-5 層嵌套的 box-shadow 創造強烈的光暈效果 (0 0 5px, 0 0 10px, 0 0 20px, 0 0 40px)
- **脈衝動畫**: 可選的持續脈衝動畫，光暈強度在 0.6-1.0 之間週期性變化
- **銳利邊緣**: 使用最小或無圓角 (border-radius: 0-2px)，強化硬邊科技感
- **對比色方案**: 極暗背景與極亮霓虹色形成強烈視覺對比

### 交互體驗

- **電流效果**: 懸停時觸發額外的光暈層和色彩飽和度提升，模擬電流增強效果
- **掃描線動畫**: 滾動時軌道上出現快速移動的掃描線效果，強化數據流動感
- **色彩切換**: 拖動時滑塊顏色可從青色切換至洋紅或其他霓虹色，增加互動趣味性
- **閃爍效果**: 滾動開始和結束時滑塊快速閃爍一次，模擬電子設備的開關狀態
- **脈衝同步**: 拖動時脈衝動畫加速，釋放後恢復正常節奏
- **殘影效果**: 可選的軌跡殘影，滑塊移動時留下短暫的光跡

### 適用場景

- **遊戲界面**: 賽博朋克主題遊戲、科幻射擊遊戲的 UI 設計
- **科技產品展示**: 前沿科技產品網站、電競硬件品牌宣傳頁
- **音樂視覺化**: 電子音樂播放器、DJ 混音平台、音頻編輯器
- **開發者工具**: 黑客主題的代碼編輯器、終端模擬器、系統監控面板
- **夜店娛樂**: 夜店活動網站、電子音樂節宣傳頁、藝術裝置互動界面
- **科幻作品**: 電影宣傳網站、科幻小說平台、未來主義藝術展示

## English Version (en-US)

Please create a Neon Cyberpunk style custom scrollbar component.

### Design Philosophy

Neon Cyberpunk scrollbar draws inspiration from cyberpunk aesthetics and neon culture, fusing futurism, urban nightscapes, and digital dystopian visual elements. The design philosophy emphasizes high contrast, glowing effects, and electronic pulse sensations, creating a visually tech-infused experience with rebellious spirit. The scrollbar is not merely a navigation tool but an energy indicator in the interface, symbolizing data flow and electronic pulsation.

### Visual Characteristics

- **Neon Glow**: Thumb uses vivid neon colors (#00ffff cyan, #ff00ff magenta, #ffff00 yellow) with box-shadow for multi-layer glow effects
- **Circuit Board Track**: Track uses dark background (#0a0a0a) with circuit texture or scanline effects, simulating electronic circuit boards
- **Multiple Halos**: Thumb uses 3-5 nested box-shadow layers creating intense glow effects (0 0 5px, 0 0 10px, 0 0 20px, 0 0 40px)
- **Pulse Animation**: Optional continuous pulse animation with glow intensity periodically varying between 0.6-1.0
- **Sharp Edges**: Minimal or no border-radius (0-2px) to reinforce hard-edge tech aesthetic
- **Contrast Color Scheme**: Extremely dark background with extremely bright neon colors forming strong visual contrast

### Interaction Experience

- **Electric Current Effect**: Hovering triggers additional glow layers and color saturation boost, simulating enhanced electrical current
- **Scanline Animation**: Fast-moving scanline effect appears on track during scrolling, reinforcing data flow sensation
- **Color Switching**: Thumb color can switch from cyan to magenta or other neon colors during drag for interactive playfulness
- **Flicker Effect**: Thumb quickly flickers once at scroll start and end, simulating electronic device on/off state
- **Pulse Sync**: Pulse animation accelerates during drag, returns to normal rhythm on release
- **Afterimage Effect**: Optional trail afterimage where thumb leaves brief light trail during movement

### Use Cases

- **Gaming Interfaces**: Cyberpunk-themed games, sci-fi shooter game UI design
- **Tech Product Showcases**: Cutting-edge tech product websites, esports hardware brand promotional pages
- **Music Visualization**: Electronic music players, DJ mixing platforms, audio editors
- **Developer Tools**: Hacker-themed code editors, terminal emulators, system monitoring dashboards
- **Nightclub Entertainment**: Nightclub event websites, electronic music festival promotional pages, art installation interactive interfaces
- **Sci-Fi Works**: Movie promotional websites, science fiction novel platforms, futurism art exhibitions
