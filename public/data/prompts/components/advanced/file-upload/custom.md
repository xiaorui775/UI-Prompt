# File Upload

## Prompt (en-US)

Create a versatile file upload component with drag-and-drop support, file type validation, and progress tracking. Feature file preview thumbnails, upload cancellation, multiple file selection, and comprehensive error handling.

---

## 提示詞 (zh-CN)

### 組件描述
文件上傳組件支持拖拽上傳和進度追蹤，適用於表單附件、圖片上傳、文檔管理等場景。

### 視覺要點
- 虛線邊框拖拽區域
- 上傳圖標和提示文字
- 文件列表顯示名稱、大小
- 上傳進度條
- 文件類型圖標區分

### 互動設計
- 拖入時邊框高亮
- 點擊觸發文件選擇器
- 上傳中顯示進度百分比
- 支持取消和重試

### 變體風格
- **默認**：清晰反饋、直觀操作、完善驗證
