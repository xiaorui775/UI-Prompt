# Focus Navigator

## Prompt (en-US)

Create an interactive focus navigator allowing users to change focus position on the webpage. Feature smooth scrolling, progress tracking, section buttons, and up/down navigation controls. Support keyboard shortcuts and visual feedback.

---

## 提示詞 (zh-CN)

### 組件描述
焦點導航器允許用戶通過點擊按鈕改變頁面焦點位置，適用於長頁面導航、章節跳轉、進度指示等場景。

### 視覺要點
- 進度指示標籤
- 章節按鈕列表
- 上/下導航箭頭
- 當前位置高亮
- 固定定位（側邊或底部）

### 互動設計
- 點擊章節按鈕跳轉
- 平滑滾動過渡
- 滾動時自動更新當前位置
- 鍵盤快捷鍵支持

### 變體風格
- **簡潔**：簡單進度標籤、章節按鈕、導航箭頭
- **側邊欄**：側邊導航、進度條、視覺反饋
