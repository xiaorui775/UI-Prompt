# Neumorphism - Table

## Prompt (en-US)

### Style Characteristics
Neumorphic tables simulate the raised and recessed textures of real objects, using same-hue light and dark shadows to create soft 3D effects. Elements appear to float above or sink into the background, offering a warm and comfortable visual experience.

### Color Scheme
- **Background**: Light gray (#E0E5EC, #F0F0F3)
- **Shadow (Light)**: rgba(255,255,255,0.8)
- **Shadow (Dark)**: rgba(0,0,0,0.15)
- **Text**: Medium gray (#5A5A5A)
- **Accent**: Soft blue or purple

### Design Details
- **Borders**: No borders, rely on shadows
- **Border Radius**: 10-15px
- **Shadow (Raised)**:
  - Outer: 5px 5px 10px #b8b9be, -5px -5px 10px #fff
- **Shadow (Inset)**:
  - Inner: inset 3px 3px 5px #b8b9be, inset -3px -3px 5px #fff
- **Spacing**: Moderate, maintain distance between elements
- **Hover**: Shadow depth changes

## 提示詞 (zh-CN)

### 風格特點
新擬物化表格模擬真實物體的凹凸質感，使用同色系明暗陰影創造柔和的立體效果。元素看起來像是從背景浮起或嵌入，觸感設計帶來溫和、舒適的視覺體驗。

### 配色方案
- **背景**：淺灰 (#E0E5EC、#F0F0F3)
- **陰影-亮**：rgba(255,255,255,0.8)
- **陰影-暗**：rgba(0,0,0,0.15)
- **文字**：中灰 (#5A5A5A)
- **強調**：柔和的藍或紫

### 細節設計
- **邊框**：無邊框，依靠陰影
- **圓角**：10-15px
- **陰影（凸起）**：
  - 外陰影：5px 5px 10px #b8b9be, -5px -5px 10px #fff
- **陰影（凹陷）**：
  - 內陰影：inset 3px 3px 5px #b8b9be, inset -3px -3px 5px #fff
- **間距**：適中，保持元素間距離感
- **懸停**：陰影深度變化
