# Glassmorphism - Table

## Prompt (en-US)

### Style Characteristics
Glassmorphic tables simulate a frosted glass texture, creating a light, modern, and tech-forward visual atmosphere. Semi-transparent elements layered on gradient backgrounds produce depth and hierarchy. Perfect for trendy tech products.

### Color Scheme
- **Background**: rgba(255,255,255,0.1) to rgba(255,255,255,0.3)
- **Borders**: rgba(255,255,255,0.2) or subtle gradient glow
- **Text**: White or dark (depending on background)
- **Base Layer**: Gradient backgrounds (purple-blue, blue-green, etc.)
- **Highlights**: Soft glow effects

### Design Details
- **Borders**: 1px solid rgba(255,255,255,0.18)
- **Border Radius**: 12-20px
- **Shadows**: 0 8px 32px rgba(0,0,0,0.1)
- **Blur**: backdrop-filter: blur(10px-20px)
- **Spacing**: Moderate padding, maintain airiness
- **Hover**: Increased background opacity, subtle glow

## 提示詞 (zh-CN)

### 風格特點
玻璃態表格模擬毛玻璃質感，營造輕盈、現代、科技感的視覺氛圍。半透明的元素層疊在漸層背景上，產生深度和層次感。適合時尚科技類產品。

### 配色方案
- **背景**：rgba(255,255,255,0.1) 至 rgba(255,255,255,0.3)
- **邊框**：rgba(255,255,255,0.2) 或微光漸層
- **文字**：白色或深色（視背景而定）
- **底層**：漸層背景（紫-藍、藍-綠等）
- **高亮**：柔和發光效果

### 細節設計
- **邊框**：1px solid rgba(255,255,255,0.18)
- **圓角**：12-20px
- **陰影**：0 8px 32px rgba(0,0,0,0.1)
- **模糊**：backdrop-filter: blur(10px-20px)
- **間距**：適中內邊距，保持透氣感
- **懸停**：背景透明度增加，輕微發光
