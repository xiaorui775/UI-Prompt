# Bootstrap 5 - Table

## Prompt (en-US)

### Style Characteristics
Bootstrap 5 tables follow mature enterprise-level design standards, providing professional out-of-the-box appearance. The styling is practical, easily recognizable, and consistent across projects—perfect for rapid development and team collaboration.

### Color Scheme
- **Primary Tone**: Bootstrap blue (#0d6efd)
- **Background**: White (#FFFFFF) alternating with light gray (#F8F9FA)
- **Borders**: Medium gray (#DEE2E6)
- **Text**: Dark gray (#212529)
- **Hover**: Light blue background (rgba(0,0,0,0.075))

### Design Details
- **Borders**: 1px solid, full border wrap
- **Border Radius**: Container 4px, inner elements no radius
- **Shadows**: None or box-shadow: 0 0.5rem 1rem rgba(0,0,0,0.15)
- **Spacing**: padding: 0.5rem (compact) or 1rem (standard)
- **Striped Rows**: Odd rows with light gray background
- **Buttons**: btn-primary blue filled buttons

## 提示詞 (zh-CN)

### 風格特點
Bootstrap 5 表格遵循成熟的企業級設計規範，提供開箱即用的專業外觀。樣式實用、易於識別、跨項目一致，適合快速開發和團隊協作。

### 配色方案
- **主色調**：Bootstrap 藍 (#0d6efd)
- **背景**：白色 (#FFFFFF) 和淺灰交替 (#F8F9FA)
- **邊框**：中灰色 (#DEE2E6)
- **文字**：深灰 (#212529)
- **懸停**：淺藍背景 (rgba(0,0,0,0.075))

### 細節設計
- **邊框**：1px solid，完整邊框包圍
- **圓角**：容器 4px，內部無圓角
- **陰影**：無或 box-shadow: 0 0.5rem 1rem rgba(0,0,0,0.15)
- **間距**：padding: 0.5rem (緊湊) 或 1rem (標準)
- **條紋行**：奇數行淺灰背景
- **按鈕**：btn-primary 藍色填充按鈕
