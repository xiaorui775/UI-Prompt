# Table

## Prompt (en-US)

Create a data table for displaying structured information in rows and columns. Feature distinct headers, alternating row colors, hover states, and pagination. Support responsive horizontal scrolling on mobile devices. Include keyboard navigation and proper ARIA labels.

---

## 提示詞 (zh-CN)

### 組件描述
表格組件將數據以行列形式清晰呈現，適用於報表、列表、訂單管理等場景。

### 視覺要點
- 表頭與數據行明確區分，表頭加粗
- 斑馬紋（交替行色）提升可讀性
- 細線邊框區分單元格
- 適當的單元格內邊距（12-16px）
- 可選：固定表頭、懸浮陰影

### 互動設計
- 行懸停背景變色，150ms 過渡
- 可排序列顯示排序圖標
- 鍵盤方向鍵導航

### 變體風格
- **極簡**：細線邊框、大量留白、灰階配色
- **Bootstrap**：標準藍色主題、條紋行、圓角
- **新粗野**：粗黑邊框、撞色、偏移陰影
- **玻璃態**：毛玻璃效果、半透明、柔和發光
- **新擬物**：凹凸陰影、淺色背景、觸感設計
