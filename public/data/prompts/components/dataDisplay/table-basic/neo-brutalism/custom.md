# Neo Brutalism - Table

## Prompt (en-US)

### Style Characteristics
Neo-brutalist tables use bold, expressive visual language that breaks traditional UI conventions. Heavy borders, striking color clashes, and intentional imperfections convey rebellion and personality. Perfect for creative and youth-oriented products.

### Color Scheme
- **Primary Tones**: Bright yellow (#FFEB3B), hot pink (#FF6B9D), cyan (#4ECDC4)
- **Borders**: Pure black (#000000)
- **Background**: White or vivid color blocks
- **Text**: Black, bold weight
- **Shadows**: Pure black offset shadows, no blur

### Design Details
- **Borders**: 2-4px solid black, no border-radius
- **Border Radius**: 0 (strictly square corners)
- **Shadows**: 4px 4px 0 #000 (hard edge offset)
- **Spacing**: Large padding, emphasize block-like feel
- **Typography**: Bold weight, larger font sizes
- **Hover**: Background color switch or shadow deepening

## 提示詞 (zh-CN)

### 風格特點
新粗野主義表格以大膽、張揚的視覺語言打破傳統 UI 的溫和審美。粗重的邊框、鮮明的撞色、刻意的不完美感，傳達出反叛和個性。適合創意類、年輕化的產品。

### 配色方案
- **主色調**：鮮黃 (#FFEB3B)、亮粉 (#FF6B9D)、天藍 (#4ECDC4)
- **邊框**：純黑 (#000000)
- **背景**：白色或鮮亮色塊
- **文字**：黑色，粗體
- **陰影**：純黑偏移陰影，無模糊

### 細節設計
- **邊框**：2-4px solid black，無圓角
- **圓角**：0（嚴格方角）
- **陰影**：4px 4px 0 #000（硬邊偏移）
- **間距**：較大內邊距，強調區塊感
- **字體**：粗體、較大字號
- **懸停**：背景色切換或陰影加深
