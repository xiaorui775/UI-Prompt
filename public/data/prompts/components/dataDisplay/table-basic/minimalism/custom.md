# Minimalism - Table

## Prompt (en-US)

### Style Characteristics
Minimalist tables pursue the "less is more" design philosophy. Remove all unnecessary decorative elements, establishing visual order through whitespace and alignment. The overall atmosphere is quiet, professional, and understated.

### Color Scheme
- **Primary Tones**: Grayscale system (#F9FAFB background, #374151 text)
- **Borders**: Ultra-light gray (#E5E7EB) or borderless
- **Highlights**: Single accent color for selected states
- **Contrast**: Maintain high contrast for readability

### Design Details
- **Borders**: 1px or borderless, rely on spacing for separation
- **Border Radius**: 0 or minimal radius (2-4px)
- **Shadows**: No shadows or very subtle shadows
- **Spacing**: Large padding (16-24px)
- **Typography**: System fonts, moderate size, ample line height
- **Hover**: Subtle background adjustment (opacity 0.05) or none

## 提示詞 (zh-CN)

### 風格特點
極簡主義表格追求「少即是多」的設計哲學。去除一切非必要的裝飾元素，通過留白和對齊建立視覺秩序。整體氛圍安靜、專業、不張揚。

### 配色方案
- **主色調**：灰階系（#F9FAFB 背景、#374151 文字）
- **邊框**：極淺灰 (#E5E7EB) 或無邊框
- **高亮**：單一強調色用於選中狀態
- **對比**：保持高對比度確保可讀性

### 細節設計
- **邊框**：1px 或無邊框，依靠間距區分
- **圓角**：0 或極小圓角 (2-4px)
- **陰影**：無陰影或極淺陰影
- **間距**：大內邊距 (16-24px)
- **字體**：系統字體、適中字號、充足行高
- **懸停**：背景微調 (opacity 0.05) 或無
