# Animated Counter

## Prompt (en-US)

Create an animated counter component that displays numerical values with smooth counting animations. Feature customizable animation duration, easing functions, and number formatting. Support start/end values, prefix/suffix text, and responsive sizing for statistics and achievement displays.

---

## 提示詞 (zh-CN)

### 組件描述
動畫計數器將數字以動態計數方式呈現，適用於統計展示、成就計數、KPI指標等需要吸引注意力的場景。

### 視覺要點
- 數字字體清晰醒目，建議使用等寬字體
- 支援前綴（如 $、¥）和後綴（如 %、+）
- 數字變化時有平滑過渡動畫
- 可選：背景卡片、圖標裝飾
- 響應式字號適配不同容器

### 互動設計
- 進入視窗時自動觸發動畫
- 動畫時長可配置（1-3秒）
- 支援緩動函數（ease-out 推薦）

### 變體風格
- **極簡**：純數字、大字號、無裝飾
- **擬物**：3D 翻牌效果、機械感
- **Material**：卡片容器、陰影層級
- **終端**：等寬字體、綠色文字、打字機效果
- **賽博朋克**：霓虹發光、故障動畫、暗色背景
