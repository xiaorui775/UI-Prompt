# Cyberpunk - Animated Counter

## Prompt (en-US)

### Style Characteristics
Cyberpunk animated counters embrace neon aesthetics, glitch effects, and futuristic digital displays. Numbers glow with vibrant neon colors against dark backgrounds, featuring chromatic aberration, scan lines, and digital distortion effects. Perfect for sci-fi interfaces, gaming dashboards, and high-tech themed applications.

### Color Scheme
- **Background**: Deep dark (#0a0a0f, #1a0a1e) with subtle purple/blue tint
- **Number (Primary Neon)**: Electric cyan (#00ffff), hot pink (#ff006e), acid green (#39ff14)
- **Number Glow**: Multiple box-shadows with neon color at varying blur radii
- **Secondary Elements**: Purple (#a855f7), magenta (#ff00ff), yellow (#ffff00)
- **Scan Lines**: rgba(0,255,255,0.05) or rgba(255,0,255,0.05)
- **Glitch Colors**: Red (#ff0000) and cyan (#00ffff) for chromatic aberration

### Design Details
- **Borders**: 1-2px neon glow border or hexagonal shapes
- **Border Radius**: Sharp angles (0-4px) or diagonal cuts for cyberpunk aesthetic
- **Shadows/Glow**:
  - Multi-layer glow: 0 0 10px, 0 0 20px, 0 0 40px, 0 0 80px neon color
  - Drop shadow: 0 4px 20px rgba(neon-color, 0.5)
- **Spacing**: Compact with focused energy, padding 12-20px
- **Typography**:
  - Font: 'Orbitron', 'Rajdhani', 'Share Tech Mono', futuristic sans-serif
  - Number: 3-5rem/700-900, letter-spacing: 0.05-0.1em, tabular-nums
  - Text-shadow: Multi-layer neon glow effect
- **Animation**:
  - Glitch: Random horizontal offset (transform: translateX), RGB split
  - Flicker: Rapid opacity changes (0.8-1) for electrical instability
  - Scan line: Animated gradient moving vertically
  - Count-up: Numbers morph with brief glitch on change
- **Effects**:
  - Chromatic aberration: Pseudo-elements with red/cyan offset
  - Grid overlay: Subtle #00ffff or #ff00ff grid lines
  - Data stream: Random characters cascading in background (Matrix-style)
- **Corner accents**: Small diagonal cuts or corner brackets in neon

## 提示詞 (zh-CN)

### 風格特點
賽博朋克動畫計數器擁抱霓虹美學、故障效果和未來數字顯示。數字在深色背景上以鮮豔霓虹色發光，具有色差、掃描線和數字失真效果。適合科幻界面、遊戲儀表板和高科技主題應用。

### 配色方案
- **背景**：深暗色 (#0a0a0f, #1a0a1e) 帶細微紫/藍色調
- **數字（主霓虹）**：電子青 (#00ffff)、熱粉 (#ff006e)、酸綠 (#39ff14)
- **數字發光**：多重 box-shadow 使用霓虹色於不同模糊半徑
- **次要元素**：紫色 (#a855f7)、洋紅 (#ff00ff)、黃色 (#ffff00)
- **掃描線**：rgba(0,255,255,0.05) 或 rgba(255,0,255,0.05)
- **故障顏色**：紅色 (#ff0000) 和青色 (#00ffff) 用於色差

### 細節設計
- **邊框**：1-2px 霓虹發光邊框或六邊形形狀
- **圓角**：銳角（0-4px）或對角切割創造賽博朋克美學
- **陰影/發光**：
  - 多層發光：0 0 10px, 0 0 20px, 0 0 40px, 0 0 80px 霓虹色
  - 投影：0 4px 20px rgba(neon-color, 0.5)
- **間距**：緊湊聚焦能量，padding 12-20px
- **字體**：
  - 字體：'Orbitron', 'Rajdhani', 'Share Tech Mono'，未來感無襯線
  - 數字：3-5rem/700-900，letter-spacing: 0.05-0.1em，tabular-nums
  - Text-shadow：多層霓虹發光效果
- **動畫**：
  - 故障：隨機水平偏移（transform: translateX），RGB 分離
  - 閃爍：快速 opacity 變化（0.8-1）模擬電氣不穩定
  - 掃描線：動畫漸變垂直移動
  - 計數：數字變形配合變化時短暫故障
- **效果**：
  - 色差：偽元素帶紅/青偏移
  - 網格疊加：細微 #00ffff 或 #ff00ff 網格線
  - 數據流：背景中隨機字符級聯（Matrix 風格）
- **邊角強調**：小型對角切割或邊角括號採用霓虹色
