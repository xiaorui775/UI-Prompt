# Minimalism - Animated Counter

## Prompt (en-US)

### Style Characteristics
Minimalist animated counters pursue clean, distraction-free number displays with smooth counting animations. Large typography with ample whitespace draws attention to the numeric value itself. Subtle easing functions create professional, natural-feeling transitions. Perfect for dashboards and metric displays where clarity is paramount.

### Color Scheme
- **Background**: White (#ffffff) or transparent
- **Text (Number)**: Deep gray (#111827 or #1f2937)
- **Text (Label)**: Medium gray (#6b7280)
- **Accent**: Optional single brand color for highlights or icons

### Design Details
- **Borders**: None
- **Border Radius**: N/A (no container)
- **Shadows**: None or very subtle text-shadow for depth
- **Spacing**: Large margin between number and label (12-16px)
- **Typography**:
  - Number: 2.5-4rem/300-400 (light weight), tabular-nums for alignment
  - Label: 0.875-1rem/400-500 uppercase or normal case
- **Animation**: Smooth count-up with easeOutQuart, duration 1000-2000ms
- **Prefix/Suffix**: Optional ($, %, +, etc.) same size or slightly smaller
- **Transition**: Numbers morph smoothly, no sudden jumps

## 提示詞 (zh-CN)

### 風格特點
極簡動畫計數器追求簡潔、無干擾的數字顯示，配合流暢的計數動畫。大字體配充足留白將注意力引導至數值本身。細膩的緩動函數創造專業、自然的過渡感。適合儀表板和指標展示，清晰度至關重要。

### 配色方案
- **背景**：白色 (#ffffff) 或透明
- **文字（數字）**：深灰 (#111827 或 #1f2937)
- **文字（標籤）**：中灰 (#6b7280)
- **強調**：可選單一品牌色用於高亮或圖標

### 細節設計
- **邊框**：無
- **圓角**：N/A（無容器）
- **陰影**：無或極細微 text-shadow 增加深度
- **間距**：數字與標籤間大間距（12-16px）
- **字體**：
  - 數字：2.5-4rem/300-400（輕字重），tabular-nums 對齊
  - 標籤：0.875-1rem/400-500 大寫或正常
- **動畫**：使用 easeOutQuart 平滑計數，持續 1000-2000ms
- **前綴/後綴**：可選（$, %, + 等）相同或略小尺寸
- **過渡**：數字平滑變形，無突然跳躍
