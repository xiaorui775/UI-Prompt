# Bootstrap 5 - Card Grid

## Prompt (en-US)

### Style Characteristics
Bootstrap 5 card grids follow enterprise-grade design standards, providing professional out-of-the-box styling for displaying collections of items. Responsive grid layouts automatically adjust column counts based on screen size using Bootstrap's 12-column system. Cards feature consistent spacing, shadows, and interaction states. Perfect for rapid development and team collaboration where standardization and responsiveness are valued.

### Color Scheme
- **Card Background**: White (#ffffff)
- **Container Background**: Light gray (#f8f9fa) or white
- **Border**: Light gray (#dee2e6)
- **Text (Heading)**: Dark gray (#212529)
- **Text (Body)**: Medium gray (#6c757d)
- **Button Primary**: Bootstrap blue #0d6efd, white text
- **Button Secondary**: Medium gray #6c757d, white text
- **Hover**: Slight border color change or shadow enhancement

### Design Details
- **Grid System**: Uses Bootstrap grid (col-md-4, col-lg-3) for responsive columns
- **Borders**: 1px solid #dee2e6 for cards
- **Border Radius**: 0.375rem (6px) for cards
- **Shadows**: 0 0.125rem 0.25rem rgba(0,0,0,0.075), hover 0 0.5rem 1rem rgba(0,0,0,0.15)
- **Spacing**:
  - Gap between cards: gutter 1.5rem (24px)
  - Card padding: 1.25rem (20px)
  - Card body padding: 1.25rem
- **Typography**:
  - Card title: 1.25rem/1.2 font-weight-500 #212529
  - Card text: 1rem/1.5 #6c757d
- **Hover**: Shadow deepens, optional transform translateY(-4px), transition 300ms ease
- **Image**: Card-img-top with border-radius on top corners only
- **Buttons**: btn-primary (0.375rem padding, 0.375rem border-radius)
- **Responsive Breakpoints**:
  - xs: 1 column
  - sm: 2 columns (col-sm-6)
  - md: 3 columns (col-md-4)
  - lg: 4 columns (col-lg-3)

## 提示詞 (zh-CN)

### 風格特點
Bootstrap 5 卡片網格遵循企業級設計標準，為顯示項目集合提供專業的開箱即用樣式。響應式網格佈局使用 Bootstrap 的 12 列系統根據屏幕尺寸自動調整列數。卡片具有一致的間距、陰影和交互狀態。適合快速開發和團隊協作，重視標準化和響應性。

### 配色方案
- **卡片背景**：白色 (#ffffff)
- **容器背景**：淺灰 (#f8f9fa) 或白色
- **邊框**：淺灰 (#dee2e6)
- **文字（標題）**：深灰 (#212529)
- **文字（正文）**：中灰 (#6c757d)
- **按鈕 Primary**：Bootstrap 藍 #0d6efd，白色文字
- **按鈕 Secondary**：中灰 #6c757d，白色文字
- **懸停**：輕微邊框色變化或陰影增強

### 細節設計
- **網格系統**：使用 Bootstrap 網格（col-md-4, col-lg-3）創建響應式列
- **邊框**：卡片 1px solid #dee2e6
- **圓角**：卡片 0.375rem (6px)
- **陰影**：0 0.125rem 0.25rem rgba(0,0,0,0.075)，懸停 0 0.5rem 1rem rgba(0,0,0,0.15)
- **間距**：
  - 卡片間隙：gutter 1.5rem (24px)
  - 卡片 padding：1.25rem (20px)
  - 卡片主體 padding：1.25rem
- **字體**：
  - 卡片標題：1.25rem/1.2 font-weight-500 #212529
  - 卡片文字：1rem/1.5 #6c757d
- **懸停**：陰影加深，可選 transform translateY(-4px)，過渡 300ms ease
- **圖片**：Card-img-top 僅頂部圓角
- **按鈕**：btn-primary（0.375rem padding，0.375rem border-radius）
- **響應式斷點**：
  - xs：1 列
  - sm：2 列（col-sm-6）
  - md：3 列（col-md-4）
  - lg：4 列（col-lg-3）
