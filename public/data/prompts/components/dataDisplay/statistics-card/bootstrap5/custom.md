# Bootstrap 5 - Statistics Card

## Prompt (en-US)

### Style Characteristics
Bootstrap 5 statistics cards follow mature enterprise design standards, providing professional out-of-the-box appearance. The styling is practical, easily recognizable, and consistent across projects—perfect for rapid development and team collaboration. Left-border color coding and gradient icon backgrounds create clear visual hierarchy.

### Color Scheme
- **Background**: White (#ffffff)
- **Left Border (Theme Colors)**:
  - Primary: #0d6efd (Bootstrap blue)
  - Success: #198754 (green)
  - Warning: #ffc107 (yellow)
  - Info: #0dcaf0 (cyan)
- **Text (Label)**: Medium gray (#6c757d)
- **Text (Value)**: Dark gray (#212529)
- **Progress Bar Background**: Light gray (#e9ecef)
- **Icon Gradients**:
  - Primary: linear-gradient(135deg, #0d6efd, #0a58ca)
  - Success: linear-gradient(135deg, #198754, #146c43)
  - Warning: linear-gradient(135deg, #ffc107, #cc9a06)
  - Info: linear-gradient(135deg, #0dcaf0, #0aa2c0)
- **Badge**:
  - Success: #198754 background, white text
  - Warning: #ffc107 background, black text
  - Info: #0dcaf0 background, black text

### Design Details
- **Borders**: Left border 4px solid (theme color), no other borders
- **Border Radius**: 0.5rem (8px)
- **Shadows**: Default 0 0.125rem 0.25rem rgba(0,0,0,0.075), hover 0 0.5rem 1rem rgba(0,0,0,0.15)
- **Spacing**: padding 1.5rem (24px), icon margin-bottom 1rem, body gap 0.5rem
- **Typography**: Label 0.875rem/500 (#6c757d), value 1.75rem/700 (#212529), meta 0.75rem (#6c757d)
- **Hover**: Shadow deepens, card translates up 3px, transition 0.3s ease
- **Progress Bar**: Height 0.5rem (8px), border-radius 0.25rem (4px), gradient fill, transition width 0.6s ease
- **Badge**: Padding 0.25rem 0.5rem, font-size 0.75rem, font-weight 600, border-radius 0.25rem

## 提示詞 (zh-CN)

### 風格特點
Bootstrap 5 統計卡片遵循成熟的企業級設計規範，提供開箱即用的專業外觀。樣式實用、易於識別、跨項目一致，適合快速開發和團隊協作。左邊框色彩編碼和漸變圖標背景創造清晰的視覺層次。

### 配色方案
- **背景**：白色 (#ffffff)
- **左邊框（主題色）**：
  - Primary：#0d6efd（Bootstrap 藍）
  - Success：#198754（綠色）
  - Warning：#ffc107（黃色）
  - Info：#0dcaf0（青色）
- **文字（標籤）**：中灰 (#6c757d)
- **文字（數值）**：深灰 (#212529)
- **進度條背景**：淺灰 (#e9ecef)
- **圖標漸變**：
  - Primary：linear-gradient(135deg, #0d6efd, #0a58ca)
  - Success：linear-gradient(135deg, #198754, #146c43)
  - Warning：linear-gradient(135deg, #ffc107, #cc9a06)
  - Info：linear-gradient(135deg, #0dcaf0, #0aa2c0)
- **徽章**：
  - Success：#198754 背景，白色文字
  - Warning：#ffc107 背景，黑色文字
  - Info：#0dcaf0 背景，黑色文字

### 細節設計
- **邊框**：左邊框 4px solid（主題色），其他無邊框
- **圓角**：0.5rem (8px)
- **陰影**：默認 0 0.125rem 0.25rem rgba(0,0,0,0.075)，懸停 0 0.5rem 1rem rgba(0,0,0,0.15)
- **間距**：padding 1.5rem (24px)，圖標 margin-bottom 1rem，主體 gap 0.5rem
- **字體**：標籤 0.875rem/500 (#6c757d)，數值 1.75rem/700 (#212529)，元數據 0.75rem (#6c757d)
- **懸停**：陰影加深，卡片上移 3px，過渡 0.3s ease
- **進度條**：高度 0.5rem (8px)，圓角 0.25rem (4px)，漸變填充，寬度過渡 0.6s ease
- **徽章**：padding 0.25rem 0.5rem，字號 0.75rem，字重 600，圓角 0.25rem
