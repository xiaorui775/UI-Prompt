# Statistics Card

## Prompt (en-US)

Create a statistics card component for presenting key metrics and KPIs. Feature title, value display, trend indicators, and comparative data visualization. Support icons, percentage changes, sparkline charts, and customizable styling for dashboard applications.

---

## 提示詞 (zh-CN)

### 組件描述
統計卡片展示關鍵指標和數據點，適用於儀表板、數據概覽、業務報表等場景。

### 視覺要點
- 主數值突出顯示（大字號、粗體）
- 標題/標籤文字較小、灰色
- 趨勢指示器（上升綠色、下降紅色）
- 可選：迷你圖表、進度條
- 支援圖標裝飾增強識別度

### 互動設計
- 懸停顯示詳細信息（tooltip）
- 點擊可展開更多數據
- 數值變化時有動畫效果

### 變體風格
- **極簡**：純文字、無邊框、留白設計
- **Bootstrap**：卡片容器、標準陰影
- **玻璃態**：半透明背景、漸層邊框
- **Ant Design**：專業配色、圖標系統
- **新擬物**：柔和凹凸、淺色背景
