# Minimalist - Statistics Card

## Prompt (en-US)

### Style Characteristics
Minimalist statistics cards pursue the "data first" design philosophy, removing all decorative elements and using large typography and whitespace to highlight core metrics. Gradient icons add visual hierarchy while trend indicators provide clear, intuitive feedback. Perfect for professional dashboards and data analysis interfaces.

### Color Scheme
- **Background**: White (#ffffff)
- **Borders**: Ultra-light gray (#e5e7eb), hover state (#d1d5db)
- **Text (Primary)**: Deep gray (#1f2937 推測)
- **Text (Secondary)**: Medium gray (#6b7280 推測)
- **Text (Meta)**: Light gray (#9ca3af 推測)
- **Trend Up**: Background #dcfce7, text #16a34a
- **Trend Down**: Background #fee2e2, text #dc2626
- **Icon Gradients**:
  - Green: linear-gradient(135deg, #10b981, #059669)
  - Purple: linear-gradient(135deg, #8b5cf6, #7c3aed)
  - Blue: linear-gradient(135deg, #3b82f6, #2563eb)

### Design Details
- **Borders**: 1px solid #e5e7eb, hover changes to #d1d5db
- **Border Radius**: 1rem (16px)
- **Shadows**: None by default, hover adds 0 10px 25px -5px rgba(0,0,0,0.1)
- **Spacing**: padding 1.5rem (24px), icon-to-value gap 1rem, trend label padding 0.25rem 0.75rem
- **Typography**: Value 3xl/font-light (1.875-2rem), label sm/font-light (0.875rem), trend 0.75rem/600
- **Hover**: Border color darkens to #d1d5db, shadow appears, card translates up 2px, transition 0.3s ease
- **Icon**: 3rem (48px) square, 0.75rem (12px) border-radius, gradient background, white icon

## 提示詞 (zh-CN)

### 風格特點
極簡統計卡片追求「數據優先」的設計哲學，去除所有裝飾元素，用大字號和留白突出核心數值。漸變圖標增添視覺層次，趨勢指示器清晰直觀。適合專業儀表板和數據分析介面。

### 配色方案
- **背景**：白色 (#ffffff)
- **邊框**：極淺灰 (#e5e7eb)，懸停狀態 (#d1d5db)
- **文字（主）**：深灰 (#1f2937 推測)
- **文字（次）**：中灰 (#6b7280 推測)
- **文字（元數據）**：淺灰 (#9ca3af 推測)
- **趨勢上升**：背景 #dcfce7，文字 #16a34a
- **趨勢下降**：背景 #fee2e2，文字 #dc2626
- **圖標漸變**：
  - 綠色：linear-gradient(135deg, #10b981, #059669)
  - 紫色：linear-gradient(135deg, #8b5cf6, #7c3aed)
  - 藍色：linear-gradient(135deg, #3b82f6, #2563eb)

### 細節設計
- **邊框**：1px solid #e5e7eb，懸停時變為 #d1d5db
- **圓角**：1rem (16px)
- **陰影**：默認無陰影，懸停時顯示 0 10px 25px -5px rgba(0,0,0,0.1)
- **間距**：padding 1.5rem (24px)，圖標與數值間距 1rem，趨勢標籤 padding 0.25rem 0.75rem
- **字體**：數值 3xl/font-light (1.875-2rem)，標籤 sm/font-light (0.875rem)，趨勢 0.75rem/600
- **懸停**：邊框變深至 #d1d5db，顯示陰影，卡片上移 2px，過渡 0.3s ease
- **圖標**：3rem (48px) 正方形，0.75rem (12px) 圓角，漸變背景，白色圖標
