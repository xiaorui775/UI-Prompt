# Neumorphism - List View

## Prompt (en-US)

### Style Characteristics
Neumorphic list views simulate raised and recessed textures of real objects, using same-hue light and dark shadows to create soft 3D effects. Each list item appears to float slightly above or press into the background surface, creating a tactile, comfortable viewing experience. The design emphasizes depth through shadows rather than borders.

### Color Scheme
- **Background**: Light gray (#e0e5ec)
- **Item Background**: Same as background (#e0e5ec), differentiated by shadow
- **Shadow (Light)**: #ffffff
- **Shadow (Dark)**: #b8bdc4
- **Text (Primary)**: Deep gray (#1f2937)
- **Text (Secondary)**: Medium gray (#6b7280)
- **Selected/Active**: Inset shadow creating pressed effect
- **Icon**: Gradient colors with soft shadows

### Design Details
- **Borders**: None, rely entirely on shadows
- **Border Radius**: 12-16px for soft, organic feel
- **Shadows (Raised)**:
  - Default: 8px 8px 16px #b8bdc4, -8px -8px 16px #ffffff
  - Hover: 10px 10px 20px #b8bdc4, -10px -10px 20px #ffffff
- **Shadows (Inset/Pressed)**:
  - Selected: inset 4px 4px 8px #b8bdc4, inset -4px -4px 8px #ffffff
- **Spacing**: padding 16-20px, gap 12-16px between items
- **Typography**: Primary text 0.875-1rem/600 #1f2937, secondary 0.75-0.875rem/500 #6b7280
- **Hover**: Shadow deepens, slight scale 1.02, transition 300ms ease
- **Icon**: 36-44px with gradient background, soft drop-shadow
- **Dividers**: None, items separated by gaps with individual shadows

## 提示詞 (zh-CN)

### 風格特點
新擬物化列表視圖模擬真實物體的凹凸質感，使用同色系明暗陰影創造柔和的 3D 效果。每個列表項看起來略微浮在背景上方或壓入背景表面，創造觸覺般舒適的觀看體驗。設計通過陰影而非邊框強調深度。

### 配色方案
- **背景**：淺灰 (#e0e5ec)
- **列表項背景**：與背景相同 (#e0e5ec)，通過陰影區分
- **陰影（亮）**：#ffffff
- **陰影（暗）**：#b8bdc4
- **文字（主）**：深灰 (#1f2937)
- **文字（次）**：中灰 (#6b7280)
- **選中/激活**：內陰影創造按壓效果
- **圖標**：漸變色配合柔和陰影

### 細節設計
- **邊框**：無邊框，完全依賴陰影
- **圓角**：12-16px 創造柔和有機感
- **陰影（凸起）**：
  - 默認：8px 8px 16px #b8bdc4, -8px -8px 16px #ffffff
  - 懸停：10px 10px 20px #b8bdc4, -10px -10px 20px #ffffff
- **陰影（凹陷/按壓）**：
  - 選中：inset 4px 4px 8px #b8bdc4, inset -4px -4px 8px #ffffff
- **間距**：padding 16-20px，項目間 gap 12-16px
- **字體**：主文字 0.875-1rem/600 #1f2937，次文字 0.75-0.875rem/500 #6b7280
- **懸停**：陰影加深，輕微縮放 1.02，過渡 300ms ease
- **圖標**：36-44px 帶漸變背景，柔和投影
- **分隔線**：無，項目通過間隙和各自陰影分離
