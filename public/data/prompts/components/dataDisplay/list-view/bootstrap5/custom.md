# Bootstrap 5 - List View

## Prompt (en-US)

### Style Characteristics
Bootstrap 5 list views follow enterprise-grade design standards with practical styling, clear visual structure, and familiar interaction patterns. List groups provide out-of-the-box styling with consistent borders, badges, and action states. Perfect for rapid development and team collaboration where standardization is valued.

### Color Scheme
- **Background**: White (#ffffff) or light gray (#f8f9fa)
- **Item Background**: White (#ffffff)
- **Border**: Medium gray (#dee2e6)
- **Text (Primary)**: Dark gray (#212529)
- **Text (Secondary)**: Medium gray (#6c757d)
- **Active Item**: Bootstrap blue #0d6efd background, white text
- **Hover**: Light gray (#e9ecef)
- **Badge Colors**:
  - Primary: #0d6efd
  - Success: #198754
  - Danger: #dc3545
  - Warning: #ffc107 with dark text
  - Info: #0dcaf0

### Design Details
- **Borders**: 1px solid #dee2e6, full border wrap for list group
- **Border Radius**: 0.375rem (6px) for container, 0 for inner items
- **Shadows**: None or subtle 0 0.125rem 0.25rem rgba(0,0,0,0.075)
- **Spacing**: padding 0.75rem 1.25rem per item
- **Typography**: Body text 1rem/1.5 #212529, secondary 0.875rem #6c757d
- **Hover**: Background #e9ecef, transition 150ms ease-in-out
- **Active**: Background #0d6efd, text white, border-color #0d6efd
- **Badge**: padding 0.35em 0.65em, font-size 0.75em, font-weight 700, border-radius 0.25rem
- **Action Items**: cursor pointer, hover darker

## 提示詞 (zh-CN)

### 風格特點
Bootstrap 5 列表視圖遵循企業級設計標準，具有實用的樣式、清晰的視覺結構和熟悉的互動模式。列表組提供開箱即用的樣式，帶有一致的邊框、徽章和操作狀態。適合快速開發和團隊協作，重視標準化。

### 配色方案
- **背景**：白色 (#ffffff) 或淺灰 (#f8f9fa)
- **列表項背景**：白色 (#ffffff)
- **邊框**：中灰 (#dee2e6)
- **文字（主）**：深灰 (#212529)
- **文字（次）**：中灰 (#6c757d)
- **激活項**：Bootstrap 藍 #0d6efd 背景，白色文字
- **懸停**：淺灰 (#e9ecef)
- **徽章顏色**：
  - Primary：#0d6efd
  - Success：#198754
  - Danger：#dc3545
  - Warning：#ffc107 帶深色文字
  - Info：#0dcaf0

### 細節設計
- **邊框**：1px solid #dee2e6，列表組完整邊框包圍
- **圓角**：容器 0.375rem (6px)，內部項目 0
- **陰影**：無或細微 0 0.125rem 0.25rem rgba(0,0,0,0.075)
- **間距**：每項 padding 0.75rem 1.25rem
- **字體**：正文 1rem/1.5 #212529，次文字 0.875rem #6c757d
- **懸停**：背景 #e9ecef，過渡 150ms ease-in-out
- **激活**：背景 #0d6efd，文字白色，邊框色 #0d6efd
- **徽章**：padding 0.35em 0.65em，字號 0.75em，字重 700，圓角 0.25rem
- **操作項**：鼠標指針，懸停更深
