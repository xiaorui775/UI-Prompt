# Minimalism Modal Dialog

## 中文版本 (zh-CN)

請創建一個極簡主義風格的模態對話框組件。

### 設計理念

極簡主義追求「少即是多」的設計哲學，通過減少視覺元素來突出核心內容。對話框應該像一張乾淨的白紙，讓信息自然呈現，不受裝飾干擾。

### 視覺特徵

**整體風格**
- 純淨的白色或淺灰色背景，無多餘裝飾
- 細緻的淺灰色邊框（1px），幾乎不可見但提供結構感
- 輕柔的陰影，僅用於區分層次，不張揚
- 小圓角設計，保持幾何感的同時增添親和力

**排版設計**
- 標題使用中等字重，不過分強調
- 內容文字採用舒適的灰色調，降低視覺壓力
- 充足的留白空間，讓內容呼吸
- 清晰的視覺層級，但過渡自然

**按鈕設計**
- 主要按鈕：純色填充，色彩克制（深灰或單一強調色）
- 次要按鈕：透明背景，僅文字顯示
- 無多餘邊框或陰影
- 懸停時微妙的顏色變化

### 交互體驗

- 淡入淡出動畫，時長短暫（150-200ms）
- 無縮放或彈跳效果，保持優雅克制
- 遮罩層透明度較低（30-40%），不過分遮擋背景
- 關閉按鈕極簡，可能僅是一個細線 X 符號

### 適用場景

- 高端品牌網站的確認對話框
- 閱讀類應用的設置彈窗
- 藝術/設計作品展示的信息提示
- 注重內容質量的產品

## English Version (en-US)

Please create a minimalist style modal dialog component.

### Design Philosophy

Minimalism pursues the "less is more" design philosophy, highlighting core content by reducing visual elements. The dialog should be like a clean sheet of paper, allowing information to present naturally without decorative interference.

### Visual Characteristics

**Overall Style**
- Pure white or light gray background with no excess decoration
- Delicate light gray border (1px), barely visible but providing structure
- Soft shadows, used only for layer differentiation, not attention-seeking
- Small border radius, maintaining geometric feel while adding approachability

**Typography Design**
- Titles use medium font weight, not overly emphasized
- Content text uses comfortable gray tones, reducing visual pressure
- Ample whitespace, letting content breathe
- Clear visual hierarchy with natural transitions

**Button Design**
- Primary button: solid fill with restrained colors (dark gray or single accent)
- Secondary button: transparent background, text only
- No extra borders or shadows
- Subtle color changes on hover

### Interaction Experience

- Fade in/out animation with short duration (150-200ms)
- No scaling or bouncing effects, maintaining elegant restraint
- Lower overlay opacity (30-40%), not overly obscuring background
- Minimalist close button, possibly just a thin-line X symbol

### Use Cases

- Confirmation dialogs for luxury brand websites
- Settings popups for reading applications
- Information prompts for art/design showcases
- Products focused on content quality
