# Neo Brutalism Modal Dialog

## 中文版本 (zh-CN)

請創建一個新粗野主義（Neo Brutalism）風格的模態對話框組件。

### 設計理念

新粗野主義大膽打破傳統設計規則，擁抱「未完成」的原始美感。強烈的對比、粗獷的邊框、直接的色彩碰撞，創造出充滿個性和衝擊力的視覺效果。對話框應該像一張貼在牆上的海報，直接、醒目、不妥協。

### 視覺特徵

**整體風格**
- 粗黑色邊框（3-4px），線條硬朗
- 無傳統陰影，改用硬邊偏移陰影
- 對角偏移的實色陰影（如右下方 6-8px 的純黑色塊）
- 直角或極小圓角，拒絕圓潤
- 高對比度配色

**色彩運用**
- 大膽使用明亮的原色：黃色、粉色、青色、綠色
- 背景可為純白或單一亮色
- 文字通常為純黑色
- 按鈕使用撞色設計

**排版設計**
- 標題可使用粗體或黑體
- 文字對齊直接，無過多修飾
- 間距可以「不規則」但有意為之
- 字體可選擇有個性的無襯線體

**按鈕設計**
- 粗邊框包裹
- 填充亮色背景（黃色、粉色等）
- 文字為黑色，大寫可選
- 帶有硬陰影效果
- 懸停時陰影消失或位移，模擬「按下」

**關閉按鈕**
- 粗大的 X 符號
- 可能帶有粗邊框圓圈或方框
- 位置可以「打破」對話框邊界

### 交互體驗

- 動畫可以誇張：彈跳、抖動
- 或者完全無動畫，直接出現
- 懸停效果明顯：色彩反轉或陰影變化
- 打破常規預期的交互反饋

### 適用場景

- 年輕潮流品牌
- 創意機構和設計工作室
- 遊戲和娛樂產品
- 追求個性和差異化的產品
- 藝術展覽和活動網站

## English Version (en-US)

Please create a Neo Brutalism style modal dialog component.

### Design Philosophy

Neo Brutalism boldly breaks traditional design rules, embracing the raw aesthetic of the "unfinished." Strong contrasts, rugged borders, direct color clashes create visual effects full of personality and impact. The dialog should be like a poster on a wall—direct, striking, uncompromising.

### Visual Characteristics

**Overall Style**
- Thick black border (3-4px), hard lines
- No traditional shadows, replaced by hard-edge offset shadows
- Diagonal offset solid shadows (e.g., 6-8px pure black block to bottom-right)
- Right angles or minimal border radius, rejecting roundedness
- High contrast color schemes

**Color Usage**
- Bold use of bright primary colors: yellow, pink, cyan, green
- Background can be pure white or single bright color
- Text typically pure black
- Buttons use contrasting color designs

**Typography Design**
- Titles can use bold or black weights
- Direct text alignment with minimal decoration
- Spacing can be "irregular" but intentional
- Fonts can be characterful sans-serifs

**Button Design**
- Wrapped in thick borders
- Filled with bright backgrounds (yellow, pink, etc.)
- Black text, uppercase optional
- Has hard shadow effect
- Shadow disappears or shifts on hover, simulating "pressed"

**Close Button**
- Thick, large X symbol
- May have thick-bordered circle or box
- Position can "break" dialog boundaries

### Interaction Experience

- Animations can be exaggerated: bounce, shake
- Or completely no animation, appearing directly
- Obvious hover effects: color inversion or shadow changes
- Interaction feedback that breaks conventional expectations

### Use Cases

- Young trend brands
- Creative agencies and design studios
- Gaming and entertainment products
- Products pursuing personality and differentiation
- Art exhibitions and event websites
