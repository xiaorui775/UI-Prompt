# Cyberpunk Modal Dialog

## 中文版本 (zh-CN)

請創建一個賽博朋克（Cyberpunk）風格的模態對話框組件。

### 設計理念

賽博朋克融合高科技與反烏托邦美學，創造出充滿未來感和頹廢感的視覺語言。霓虹燈光、數字故障、電路紋理，對話框應該像是來自 2077 年的全息投影界面。

### 視覺特徵

**整體風格**
- 深色背景（接近黑色的深藍、深紫）
- 霓虹色邊框和高亮元素（青色、粉色、紫色、黃色）
- 發光效果：邊框和文字帶有外發光（glow）
- 可選：掃描線或噪點覆蓋效果
- 斜切角設計（corner cuts）替代圓角

**霓虹效果**
- 邊框帶有多層發光（box-shadow）
- 文字可帶有輕微發光效果
- 關鍵元素使用高飽和度霓虹色
- 發光效果可添加動畫（呼吸、閃爍）

**背景處理**
- 半透明深色背景
- 可選：電路板紋理、網格線、六邊形圖案
- 可選：輕微的故障效果（glitch）

**排版設計**
- 可使用等寬字體或科技感字體
- 文字顏色使用淺色（白色或霓虹色）
- 可添加「系統提示」風格的前綴
- 標題可使用大寫字母

**按鈕設計**
- 邊框發光效果
- 透明或半透明背景
- 懸停時發光增強
- 可選：邊框動畫（掃描線效果）
- 激活時顏色變化明顯

### 交互體驗

- 故障抖動效果出現
- 掃描線動畫
- 霓虹燈閃爍效果
- 打字機效果的文字顯示（可選）
- 聲效提示的視覺暗示

### 裝飾元素

- 角落的裝飾框架
- HUD（抬頭顯示）風格的信息標籤
- 假的載入指示器或進度條
- 數據流或代碼雨背景

### 適用場景

- 遊戲相關網站和應用
- 科幻主題產品
- 電子音樂和夜店活動
- 極客和黑客文化社區
- NFT 和 Web3 產品

## English Version (en-US)

Please create a Cyberpunk style modal dialog component.

### Design Philosophy

Cyberpunk merges high-tech with dystopian aesthetics, creating visual language full of futurism and decadence. Neon lights, digital glitches, circuit textures—the dialog should appear like a holographic projection interface from 2077.

### Visual Characteristics

**Overall Style**
- Dark background (deep blue or purple close to black)
- Neon-colored borders and highlights (cyan, pink, purple, yellow)
- Glow effects: borders and text with outer glow
- Optional: scanlines or noise overlay effects
- Corner cuts design replacing rounded corners

**Neon Effects**
- Borders with multiple layers of glow (box-shadow)
- Text can have subtle glow effect
- Key elements use high-saturation neon colors
- Glow effects can include animations (breathing, flickering)

**Background Treatment**
- Semi-transparent dark background
- Optional: circuit board texture, grid lines, hexagon patterns
- Optional: subtle glitch effect

**Typography Design**
- Can use monospace or tech-style fonts
- Light text colors (white or neon)
- Can add "system prompt" style prefixes
- Titles can use uppercase letters

**Button Design**
- Border glow effects
- Transparent or semi-transparent backgrounds
- Enhanced glow on hover
- Optional: border animations (scanline effect)
- Obvious color change on activation

### Interaction Experience

- Glitch-shake effect on appearance
- Scanline animations
- Neon light flicker effects
- Typewriter effect for text display (optional)
- Visual hints suggesting sound effects

### Decorative Elements

- Corner decorative frames
- HUD (heads-up display) style info labels
- Fake loading indicators or progress bars
- Data stream or code rain backgrounds

### Use Cases

- Gaming related websites and apps
- Sci-fi themed products
- Electronic music and nightclub events
- Geek and hacker culture communities
- NFT and Web3 products
