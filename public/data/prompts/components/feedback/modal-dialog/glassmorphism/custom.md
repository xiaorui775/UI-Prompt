# Glassmorphism Modal Dialog

## 中文版本 (zh-CN)

請創建一個玻璃態（Glassmorphism）風格的模態對話框組件。

### 設計理念

玻璃態設計模擬真實世界中磨砂玻璃的質感，創造出半透明、有層次、現代感十足的視覺效果。對話框應該像一塊懸浮的毛玻璃，透過它能隱約看到背後的世界。

### 視覺特徵

**整體風格**
- 半透明白色或淺色背景（60-80% 透明度）
- 強烈的背景模糊效果（blur 12-20px）
- 細膩的白色或淺色邊框，增強玻璃邊緣感
- 中到大圓角，營造柔和的玻璃形態
- 可選：微妙的內發光效果模擬光線折射

**層次效果**
- 多層玻璃堆疊時，透明度和模糊度遞減
- 遮罩層保持較高透明度，讓背景色彩透出
- 玻璃表面可添加輕微的漸變，模擬光線變化

**排版設計**
- 文字需要足夠對比度以保證可讀性
- 標題可使用較重字重或深色
- 可考慮文字陰影增強層次感

**按鈕設計**
- 主按鈕：半透明填充或實色填充
- 次要按鈕：更透明的玻璃效果
- 懸停時透明度或亮度變化
- 邊框高亮效果增強交互反饋

### 交互體驗

- 平滑的模糊過渡動畫
- 透明度漸變配合位移動畫
- 懸停時玻璃效果微妙變化（更亮或更清晰）
- 點擊時可添加漣漪或光暈效果

### 色彩建議

- 背景建議使用漸變色或有色彩的圖片
- 對話框玻璃可略帶色調（如淡藍、淡紫）
- 邊框使用白色或淺色帶透明度

### 適用場景

- 現代科技產品界面
- 具有豐富背景圖像的網站
- iOS/macOS 風格的 Web 應用
- 追求現代感和高級感的產品

## English Version (en-US)

Please create a Glassmorphism style modal dialog component.

### Design Philosophy

Glassmorphism simulates the texture of frosted glass in the real world, creating semi-transparent, layered, and modern visual effects. The dialog should appear like a floating frosted glass panel, through which you can vaguely see the world behind.

### Visual Characteristics

**Overall Style**
- Semi-transparent white or light-colored background (60-80% opacity)
- Strong background blur effect (blur 12-20px)
- Delicate white or light border enhancing glass edge perception
- Medium to large border radius creating soft glass form
- Optional: subtle inner glow simulating light refraction

**Layering Effects**
- When stacking multiple glass layers, opacity and blur decrease progressively
- Overlay maintains high transparency to let background colors show through
- Glass surface can include subtle gradients simulating light changes

**Typography Design**
- Text needs sufficient contrast for readability
- Titles can use heavier weights or darker colors
- Consider text shadows for enhanced depth

**Button Design**
- Primary button: semi-transparent or solid fill
- Secondary button: more transparent glass effect
- Opacity or brightness changes on hover
- Border highlight effects enhance interaction feedback

### Interaction Experience

- Smooth blur transition animations
- Opacity gradients combined with displacement animation
- Subtle glass effect changes on hover (brighter or clearer)
- Can add ripple or halo effects on click

### Color Recommendations

- Background should use gradients or colorful images
- Dialog glass can have subtle tints (light blue, light purple)
- Borders use white or light colors with transparency

### Use Cases

- Modern tech product interfaces
- Websites with rich background imagery
- iOS/macOS style web applications
- Products pursuing modern and premium aesthetics
