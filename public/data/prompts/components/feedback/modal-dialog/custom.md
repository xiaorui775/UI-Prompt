# Modal Dialog

## Prompt (en-US)

Create a modal dialog for displaying important information or requesting user actions. Feature customizable content, action buttons, backdrop overlay, and smooth open/close animations. Support keyboard navigation, focus trapping, and ESC to close.

---

## 提示詞 (zh-CN)

### 組件描述
模態對話框在覆蓋層中顯示重要內容或請求用戶操作，適用於確認提示、表單填寫、信息展示等場景。

### 視覺要點
- 居中顯示的白色/主題色容器
- 半透明暗色背景遮罩
- 標題欄、內容區、操作按鈕區
- 關閉按鈕（X 或取消）
- 可選：圓角、陰影、邊框

### 互動設計
- 點擊遮罩可關閉（可配置）
- ESC 鍵關閉
- 焦點鎖定在對話框內
- 淡入/滑入動畫

### 變體風格
- **極簡**：簡潔樣式、乾淨邊框
- **Material**：提升陰影、波紋效果
- **玻璃態**：毛玻璃背景
- **新擬物**：柔和凹凸陰影
- **Bootstrap**：標準 Bootstrap 樣式
- **新粗野**：粗邊框、撞色
- **賽博朋克**：霓虹發光、科技感
- **底部抽屜**：從底部滑出
- **側邊抽屜**：從側邊滑出
- **通知**：通知樣式對話框
- **圖片預覽**：圖片展示專用
