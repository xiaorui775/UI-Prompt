# Reaction Picker

## Prompt (en-US)

Create an interactive reaction picker for emoji or icon selection. Feature emoji library with search, recent reactions tracking, and customizable categories. Support smooth hover animations, keyboard navigation, and visual selection feedback.

---

## 提示詞 (zh-CN)

### 組件描述
表情反應選擇器允許用戶選擇 emoji 或反應圖標，適用於社交互動、評論反饋、即時通訊等場景。

### 視覺要點
- 表情網格排列
- 分類標籤（常用、笑臉、動物等）
- 搜索輸入框
- 選中狀態高亮
- 彈出式容器

### 互動設計
- 懸停時表情放大
- 點擊選中表情
- 支援鍵盤方向鍵導航
- 最近使用記錄

### 變體風格
- **Material**：圓潤卡片、波紋效果
- **玻璃態**：半透明毛玻璃
- **新擬物**：柔和浮雕效果
- **極簡**：簡潔黑白設計
