# Loading Animate

## Prompt (en-US)

Create versatile loading animations with multiple variants: spinner, dots, progress bar, skeleton, pulse, wave, ring, and bounce. Feature smooth animations, customizable colors and sizes, and accessibility support for reduced motion preferences.

---

## 提示詞 (zh-CN)

### 組件描述
載入動畫組件提供多種載入狀態指示，適用於頁面載入、數據請求、按鈕等待等場景。

### 視覺要點
- 動畫流暢無卡頓
- 大小可配置
- 顏色可自定義
- 居中顯示或內聯

### 互動設計
- 尊重 prefers-reduced-motion
- 可選：載入文字提示
- 可選：進度百分比

### 變體風格
- **旋轉器**：經典圓形旋轉
- **三點**：彈跳的三個點
- **進度條**：線性進度指示
- **骨架屏**：內容佔位動畫
- **脈衝**：大小脈動效果
- **波浪**：依次起伏的條形
- **圓環**：邊框旋轉
- **彈跳球**：彈跳的圓球
