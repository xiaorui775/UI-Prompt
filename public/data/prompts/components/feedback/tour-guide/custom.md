# Tour Guide

## Prompt (en-US)

Create a step-by-step user guidance component for displaying product features and workflows. Feature smart highlighting, step navigation, interactive tooltips, and progress indicators for effective onboarding and feature introduction.

---

## 提示詞 (zh-CN)

### 組件描述
導覽指南以步驟方式引導用戶了解產品功能和操作流程，適用於新用戶引導、功能介紹、操作教學等場景。

### 視覺要點
- 遮罩層突出目標元素
- 提示框顯示說明文字
- 步驟進度指示（如 1/5）
- 上一步/下一步按鈕
- 跳過按鈕

### 互動設計
- 點擊按鈕切換步驟
- 目標元素聚光燈效果
- 平滑過渡到下一目標
- 可按 ESC 跳過

### 變體風格
- **默認**：步驟引導、聚光燈效果、平滑過渡
